过滤野值¶

丢弃值超过最大值的某个百分比的所有项

例如只保留 count_piecies 列中 10% ~ 90% 区间的值（基于最大值计算）

In [4]:
# 按比例过滤 outliers
mask1 = df['count_piecies'] < np.percentile(df['count_piecies'], 90)
mask2 = df['count_piecies'] > np.percentile(df['count_piecies'], 10)

df.loc[mask1 & mask2]

merge 细节¶

使用 merge 时，需要注意两个参数：

  • suffixes (str, str) tuple, 默认 (‘_x’, ‘_y’)，用于区分两个 DataFrame 中均存在的 columns
  • indicator 布尔值，默认 False。若为 True 则添加一列 _merge，展示该行来自于哪个 DataFrame

相关的函数还有 concat

In [12]:
left = pd.DataFrame({'key': ['foo', 'foo'], 'value': [1, 1], 'lval': [1, 2]})
right = pd.DataFrame({'key': ['foo', 'bar'], 'value': [2, 3], 'rval': [4, 5]})

display(left)
display(right)

0 foo 1 1
1 foo 1 2
key value rval
0 foo 2 4
1 bar 3 5
In [13]:
pd.merge(left, right, on='key', how="inner",
         suffixes=("_left", "_right"), indicator=True)
